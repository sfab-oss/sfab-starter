# ALW-594 Template-init interview v2 — intake contract: answers → product brief → reshape task slate

## Context
`docs/template-init.md` ([[ALW-461]]) defines a v1 interview — 8 product-owner questions plus an answers→codebase-actions translation table — addressed to a coding agent adopting the template after fabrication. The platform is designing a chat-first intake ([[ALW-591]]) that runs an interview with the user before/at fabrication. The two sides must share one contract: what the interview produces, where it persists, and how it seeds the initial reshape tasks. It is a single-commit v1 that has never been iterated or battle-tested.

## Scope
- Iterate the question set with a conversational platform intake in mind: ordering, one-question-at-a-time flow, smallest-win emphasis, and stop conditions (enough to start ≠ complete spec).
- Define the durable "product brief" artifact the interview produces (shape + where it lives), so both the platform intake and the in-repo reshape agent read the same brief.
- Map brief → initial reshape task slate using the existing translation table, concretely enough that tasks can be generated mechanically.
- Decide (with [[ALW-591]]) whether `template-init.md` stays the single source the platform consumes, or the platform owns a copy — record the decision either way.

## Acceptance criteria
- [ ] AC-1: Revised interview in `docs/template-init.md` (or successor doc) usable both by an adopting agent and a conversational intake.
- [ ] AC-2: Product-brief artifact defined (shape, location, lifecycle) and referenced from the doc.
- [ ] AC-3: Brief→task-slate mapping documented with at least one worked example.
- [ ] AC-4: Source-of-truth decision recorded and cross-referenced with [[ALW-591]].

## Non-goals
- Building any platform surface (that's [[ALW-591]]'s subtasks).
- Capability-pack mechanics (ADR-0017 / [[ALW-350]]).

## Boundaries
- ✅ Always: keep the doc honest about what exists in the repo today.
- ⚠️ Ask first: restructuring docs/ layout.
- 🚫 Never: reference features or packs the template does not ship.
