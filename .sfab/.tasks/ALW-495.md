# ALW-495 Starter: own Biome ruleset, drop Ultracite, bump Biome

## Context

Closes the deferred half of starter D8 / ALW-326 AC-6: stop extending Ultracite; own the Biome config; bump Biome. Parent Group: [[ALW-494]]. Platform port is a follow-on deliverable under the same Group.

## Scope

- Remove `ultracite` dependency; `biome.jsonc` must not `extends` Ultracite.
- **Fork strategy (settled):** copy Ultracite's `core` + `react` presets into a repo-owned config (drop `next` — we are TanStack Start, not Next). Use the local copy to analyze keep/trim; do not stay on Ultracite as a runtime dependency.
- Bump `@biomejs/biome` to current latest at implementation (today 2.5.x); fix fallout.
- Preserve intentional starter overrides (e.g. `noBarrelFile: off` for capability barrels).
- Add first smell-oriented lint gates where Biome can enforce them (details settled in sharpening).
- `pnpm lint:check` / typecheck / test green.

## Acceptance criteria

- [ ] AC-1: `ultracite` gone from package.json / lockfile; `biome.jsonc` has no ultracite extend. verify: grep + `pnpm lint:check`
- [ ] AC-2: `@biomejs/biome` bumped to agreed target; lint passes on the forked ruleset. verify: package.json + `pnpm lint:check`
- [ ] AC-3: Intentional overrides preserved (barrels, route filename convention, etc.). verify: inspect `biome.jsonc`
- [ ] AC-4: Documented smell lint gates from the sharpening plan are present (or explicitly deferred with reason). verify: `biome.jsonc` + smell doc/guide cross-link
- [ ] AC-5: Full gate suite green. verify: `pnpm typecheck && pnpm lint:check && pnpm test`
