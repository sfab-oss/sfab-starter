# ALW-695 Audit biome-ignored useEffects — remove or sharpen each justification

## Context

[[ALW-672]] landed the Biome GritQL gate that bans casual `useEffect` / `useLayoutEffect` outside vendored shadcn. Existing call sites were justified with `biome-ignore` so lint stayed green — that was a **gate**, not a cleanup. Many ignores still say "external sync — revisit per code-smells.md"; this task is that revisit.

## Scope

- Inventory every `biome-ignore lint/plugin/no-use-effect` / `no-use-layout-effect` (and any remaining bare effects if any slip through) under `apps/` + `packages/` (excluding `packages/ui/src/components/shadcn/**`).
- For each site: challenge whether it is true external sync (subscription, timer, imperative DOM/3rd-party, matchMedia, etc.) vs prop/query→state mirroring.
- Prefer **remove** (derive during render, RHF `values`, event handlers) over keeping the ignore. Keep ignore only with a sharper, site-specific reason.
- Ground decisions in `docs/engineering/code-smells.md` §1; extend that doc with starter-specific fixed references when a removal lands.

## Acceptance criteria

- [ ] AC-1: Written inventory of every ignore/effect site with verdict: remove / keep (reason) / defer (why). verify: artifact or PR description table
- [ ] AC-2: All "remove" verdicts implemented; `pnpm lint:check` still green with the gate on.
- [ ] AC-3: Remaining ignores have concrete reasons (not the generic "revisit" placeholder).
- [ ] AC-4: `docs/engineering/code-smells.md` updated with at least one starter-local "fixed" reference if any removal shipped.

## Non-goals

- Enabling `noConsole` / structured logging.
- Changing the GritQL plugin itself or shadcn carve-out.
- Drive-by refactors unrelated to effects.

## Boundaries

- ✅ Always: challenge each site against React's "You Might Not Need an Effect"; keep behavior identical when removing.
- ⚠️ Ask first: removals that change UX timing (focus, scroll, mount animations) in a user-visible way.
- 🚫 Never: blanket-delete ignores to force "green" without a replacement pattern; disable the lint gate.
