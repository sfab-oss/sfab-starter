- **Filter toolbar** ‚Äî column filter popover, chips, row count (`filterDefinitions`)
- **Sort control** ‚Äî multi-column sort popover + `SortableHeader` on columns
- **Row actions** ‚Äî inline CTA and kebab menu
- **Embedded mode** ‚Äî no outer border when inside `ShellContent`
