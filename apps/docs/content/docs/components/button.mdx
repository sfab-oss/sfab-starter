The `Button` is a `registry:ui` entry. Its demo lives at
`apps/docs/src/registry/demos/button-demo.tsx` and is registered by name in
`apps/docs/src/registry/index.ts`, so `<ComponentPreview>` resolves it — the same
map that powers the full-screen `/view/button` route.

<ComponentPreview name="button" />

## Usage

```tsx
import { Button } from "@workspace/ui/components/shadcn/button";

<Button variant="outline" size="sm">
  Click me
</Button>;
```

## Variants

`default` · `secondary` · `outline` · `ghost` · `destructive` · `link`.

## Sizes

`sm` · `default` · `lg` · `icon`.
