The `Button` is a `registry:ui` entry. Its demo lives at
`packages/registry/registry/button/button-demo.tsx` and is surfaced through the
generated `@workspace/registry` map, so `<ComponentPreview>` resolves it by name —
the same map that powers the full-screen `/view/button` route.

<ComponentPreview name="button" />

## Usage

```tsx
import { Button } from "@workspace/ui/components/shadcn/button";

<Button variant="outline" size="sm">
  Click me
</Button>;
```

## Variants

`default` · `secondary` · `outline` · `ghost` · `destructive` · `link`.

## Sizes

`sm` · `default` · `lg` · `icon`.
