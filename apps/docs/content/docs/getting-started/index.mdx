## Prerequisites

- Node.js 20+
- pnpm 10+
- A Cloudflare account (for deployment)

## Install

From the monorepo root:

```bash
pnpm install
```

## Dev servers

```bash
pnpm dev
```

- Web app: [http://localhost:4011](http://localhost:4011)
- Docs app: [http://localhost:4012](http://localhost:4012)

## Next steps

Read `README.md` and `AGENTS.md` at the repo root for architecture conventions, database commands, and how to extend the template.
